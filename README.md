# Kotlin Assignment 1

This repository contains my first Kotlin programming assignment for the **MWD3B Android Development** course.

The project demonstrates fundamental Kotlin concepts, including functions, loops, lambda expressions, recursion, input validation, and console output.

## Assignment Challenges

# Challenge A: Sum of Integers

Creates a function that calculates the sum of all integers beginning at `1` and ending at a specified maximum value.

Example:


1 + 2 + 3 + 4 + 5 = 15


# Challenge B: Iterative Factorial

Creates an iterative function that calculates the factorial of a specified non-negative integer using a loop.

Example:


5! = 5 × 4 × 3 × 2 × 1 = 120


#  Challenge C: Factorial Using a Lambda

Rewrites the iterative factorial calculation as a Kotlin lambda expression.

The lambda accepts an integer and returns its factorial as a `Long` value.

# Challenge D: Recursive Factorial

Creates a recursive function that calculates the factorial of a specified non-negative integer.

The function repeatedly calls itself with a smaller value until it reaches the base case of `0` or `1`.
